# ImagesProcessUI
A C# sample for test the DLL functions provided by C++.
## How to build
### Windows
1. Download visual studio 2022(recommonded)
2. Open the this repo via visual studio 2022 as a solution
3. Build this project using `Ctrl + shift + B`
